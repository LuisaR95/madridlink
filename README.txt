# MADRILINK - Gestión de Trámites Administrativos
Proyecto Final de Ciclo - DAM (Desarrollo de Aplicaciones Multiplataforma)

## Requisitos Previos
- Java JDK 21 o superior instalado.
- Servidor MySQL 8 activo.
- Base de datos 'madridlink' creada (los datos se cargan automáticamente desde import.sql).

## Ejecución del Proyecto
1. Abrir la carpeta /target en una terminal.
2. Ejecutar el comando:
   java -jar madridlink-0.0.1-SNAPSHOT.jar
3. Acceder en el navegador a: http://localhost:10000

## Funcionalidades Principales
- Registro y Login de usuarios (Spring Security).
- Gestión interactiva de trámites con barra de progreso.
- Directorio de sedes administrativas.
- Espacio de comunidad (Muro de consejos).
